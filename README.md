# GitHub Enterprise ポータル

社内の GitHub Enterprise を利用するメンバー向けのポータルサイトです。
GitHub Enterprise の概要・主な機能・はじめかた・よくある質問をまとめています。

## 構成

```
github-enterprise-portal/
├── index.html   # ポータルのメインページ
└── style.css    # スタイルシート
```

## コンテンツ

- **GitHub Enterprise とは** — プラットフォームの概要とメリット
- **主な機能** — リポジトリ管理・PR・Actions・Packages など
- **はじめかた** — 新規メンバー向けのセットアップ手順（4ステップ）
- **よくある質問** — よくある疑問への回答

## 利用方法

`index.html` をブラウザで開くか、GitHub Pages / 社内 Web サーバーにデプロイして利用してください。
